    Welcome

    The Sky Starter WordPress Theme is a starter theme that can be used as a starting point for any custom theme. It is based of the famous Twitter Bootstrap to provide simple and flexible HTML, CSS, and Javascript for popular user interface components and interactions. It also provides other various helper functions to aide in the development of custom WordPress themes.
